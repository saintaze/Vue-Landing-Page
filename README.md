# Vue-Landing-Page
An app that displays landing page with tabs view  

### Features
+ Beautiful UI
+ Reusable Tabs and Tab Component
##### Built With
+ HTML5
+ SASS
+ Vue 3
+ Vuex

### Getting Started
clone repo

`cd` into folder and run `npm install`

run `npm run serve`
